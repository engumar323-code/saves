Important Notes

Demo System: This is a demonstration/educational system suitable for learning and testing. For production use with real money, implement proper security measures including server-side processing, authentication, encryption, and compliance with financial regulations.
  • Data persists only in your browser's local storage
  • Clearing browser cache will delete all data - regular backups recommended
  • Export transactions regularly for backup
  • Single-user system (no multi-user support yet)
  • Not suitable for real financial transactions without security upgrades

Common Questions

How do I start using the system?

Simply open index.html in your web browser. The system loads with sample data that you can use to explore all features.

Where is my data stored?

All data is stored in your browser's local storage. It persists between sessions unless you clear your browser cache.

How do I backup my data?

Go to the Transactions page and click "Export" to download all transactions as a CSV file. Save this file for backup.

Can multiple people use this?

This version is single-user. All users would share the same data. For multi-user support, upgrade to the enterprise version.
